Introduction

This is my course diary for the University of Helsinki course Data Visualization in R for PhD students. Ill use this diary to record (mainly for myself) my progress on the course by including relevant exercises and assessments. Additionally I hope to use this diary to facilitate my own reflections and thoughts about data visualization more generally.

Below are my previous experience and expectations for the course. This was part of home assignment on the first lecture.

Previous experience

Courses on:

  • Basics of programming (Java)
  • Scientific computing (Python, Matlab and C)
  • Introduction to Open Data Science (R)
  • Machine learning (Python)

Additionally, I have done two simple online tools with Javascript that are used in research and in teaching. Actually, my lesson observation tool includes simple timeline visualization of observations that is drawn on HTML-canvas using only Javascript. It was rather interesting to figure out how to draw axes, ticks, labels and the data with just canvas methods.

I have mainly used Python in my analyzes and data visualization (Masters thesis, two articles and few posters etc.). I plan to use R more in the future.

Expectations

I think I have quite good basic understanding of data visualization, but through this course I hope to refine and expand my knowledge and skills. I want to practice and become more proficient with R and ggplot2 as those have some benefits over Python. Additionally, there are some nice network analysis packages for R that I might need to use in the future.

Week 1

The first week had only three very basic exercises with base R and I completed with one simple r-script file. Thus, the first week exercises are not included in this course diary. However, I did set up github repository and this course diary webpage. Additionally, I loaned the course book Data Visualization: A Practical Introduction by Kieran Healy and The functional art: : An introduction to information graphics and visualization by Alberto Cairo. At a glance both seem interesting and useful.

Exercise 3

3.1

Calculate the mean, standard deviation (sd), and number of observations (n) of score by education and age group. Also calculate the standard error of the mean (by using sd and n). Save these into a new data frame (or tibble) named edu_age_stats.

edu_age_stats <- df %>% group_by(education, age_group) %>% summarise(mean=mean(score), sd=sd(score), n=n())
## `summarise()` has grouped output by 'education'. You can override using the `.groups` argument.
# Compute standard error of mean as sd/sqrt(n)
edu_age_stats$se = edu_age_stats$sd/sqrt(edu_age_stats$n)

3.2

Using edu_age_stats, plot a bar plot that has age group on the x-axis, mean score on the y-axis, and education levels in subplots (facets).

Use geom_errorbar() to add error bars that represent standard errors of the mean.

edu_age_stats %>% ggplot(aes(x = age_group, y = mean)) +
  facet_wrap("education", labeller = "label_both") +
  geom_col(fill = "#fcd116") + # Color of our faculty for fun
  geom_errorbar(aes(ymin=mean-se, ymax=mean+se), colour = "#002b7f") +
  theme_classic() +
  theme(axis.text.x = element_text(angle = 45, hjust = 1, size = 8)) +
  scale_y_continuous(name = "Mean score", limits = c(0, 20)) +
  scale_x_discrete(name = "Age Group")
